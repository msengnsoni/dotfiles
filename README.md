# dotfiles 設定ファイルまとめ

* .vimrc
$HOME/.vimrc

* .gitconfig
$HOME/.gitconfig

* .gitignore_global
$HOME/.gitignore_global

* .zshrc
$HOME/.zshrc

* iterm2
    1. com.googlecode.iterm2.plist
    $HOME/dotfiles/iterm2 に置く
    1. Japanesque.itermcolors
    Preference > Profiles > Colors > Color Presets > Import

* Brewfile
brew bundle でインストール

* karabiner.json
$HOME/.config/karabiner/karabiner.json
